# 📘 试卷处理智能助手使用说明

---

## 🧠 一、系统简介

本系统是一个**自动试卷处理助手**，用于帮助老师快速完成以下工作：

* 📄 试卷切题（自动分割每一道题）
* ✍️ 答案解析提取
* 🧩 知识点识别与整理
* 📊 自动生成 Excel（tags.xlsx）
* 📦 自动打包整理资料
* ☁️ 上传至飞书云盘并生成访问链接

👉 你只需要上传试卷文件，系统会自动完成后续所有步骤。

---

## 🚀 二、标准使用流程（强烈建议按此操作）

### Step 1️⃣ 上传试卷文件

请上传以下文件：

* ✅ 空白试卷 PDF（必须）
* ✅ 答案解析 PDF（必须）

📌 支持：

* 一次上传一个文件
* 或一次上传多个文件（推荐）

---

### Step 2️⃣ 确认材料

系统会提示：

> 已接收文件，是否开始处理？

请回复：

* ✅ **确认 / 开始 / 可以**
* ❌ 如果文件不对，请重新上传

---

### Step 3️⃣ 系统自动处理（无需操作）

系统会自动执行以下步骤：

1. ✂️ 试卷结构解析与切题
2. 🧠 答案与知识点提取
3. 📊 Excel 表格生成（tags.xlsx）
4. 📦 文件整理与打包
5. ☁️ 上传飞书云盘

📢 过程中你会看到进度提示，例如：

* 正在切题...
* 正在解析答案...
* 正在生成 Excel...
* 正在上传结果...

⏳ 一般耗时：1～3分钟（取决于试卷复杂度）

---

### Step 4️⃣ 获取结果

处理完成后，你会收到：

* 📁 交付文件夹名称
* 📦 文件列表
* 🔗 飞书云盘访问链接

👉 点击链接即可查看和下载所有结果

---

## 🔁 三、重新处理功能（非常重要）

当出现以下情况时，可以使用“重新处理”：

* ❌ 处理失败
* 📉 解析结果不理想
* 📊 Excel 有问题
* 📦 打包或上传异常

👉 **无需重新上传文件！**

---

### 支持的重新处理操作

| 指令         | 作用           |
| ---------- | ------------ |
| 重新分析答案和知识点 | 只重新做解析       |
| 重新生成 Excel | 只重做表格        |
| 重新打包       | 重新整理文件       |
| 重新上传       | 重新上传云盘       |
| 重新处理       | 从头开始重新执行整条流程 |

---

### 推荐使用方式

#### 情况 1：只是某一步有问题

👉 优先使用“单项重跑”

例如：

* 解析不准 → `重新分析答案和知识点`
* Excel不对 → `重新生成 Excel`

---

#### 情况 2：不确定哪里出问题

👉 使用：

```id="reprocess_example"
重新处理
```

系统会：

* 从切题开始重新执行
* 覆盖原流程结果
* 自动重新生成最终交付

---

| 指令         | 功能     |
| ---------- | ------ |
| 当前任务状态     | 查看当前进度 |
| 当前任务结果     | 获取结果链接 |
| 重新处理       | 从头重新执行 |
| 重新分析答案和知识点 | 重跑解析   |
| 重新生成 Excel | 重做表格   |
| 重新打包       | 重打包    |
| 重新上传       | 重传结果   |

## 📄 五、文件要求

### 支持格式

* ✅ PDF 文件

---

### 文件规范建议

为了保证处理效果，请尽量满足：

* 空白试卷和解析试卷内容一致
* 图片清晰，无严重模糊
* 页面完整（不要缺页）
* 题号清晰（如 1、2、3...）

---

## ⚠️ 六、常见问题（非常重要）

---

### ❓1. 为什么处理失败了？

可能原因：

* OCR服务异常（如识别服务暂时不可用）
* PDF质量较差（模糊、倾斜）
* 文件内容不规范
* 网络或上传异常

---

### ❓2. 出现“处理失败”怎么办？

建议：

1. 优先尝试 👉 `重新处理`
2. 如果仍失败：

   * 重新上传文件
   * 检查文件质量

---

### ❓3. 为什么没有结果？

如果你看到：

> 当前任务执行失败，因此暂时没有可获取的处理结果

👉 说明任务中途出错，系统已自动停止（正常行为）

---

### ❓4. 系统会卡住吗？

不会。

👉 如果出现问题，系统会：

* 自动停止
* 明确提示错误原因

## 🎯 八、使用建议（提升效率）

* 📌 一次上传完整材料，减少反复操作
* 📌 出现问题优先用“重新处理”
* 📌 尽量避免重复上传同一份文件
* 📌 善用单项重跑，效率更高
